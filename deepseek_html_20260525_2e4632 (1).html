📖 Python 类与对象 · 核心知识点

🏷️ 类与对象定义

类是创建对象的蓝图，对象是类的实例。通过 __init__ 构造方法初始化属性。

class Book: def __init__(self, title, author): self.title = title self.author = author def info(self): print(f"《{self.title}》的作者是{self.author}") book = Book("Python编程", "Eric") book.info()

💰 封装与继承

封装隐藏内部细节，继承实现代码复用。

class Student: def __init__(self, name): self.name = name class GraduateStudent(Student): def __init__(self, name, area): super().__init__(name) self.research_area = area
